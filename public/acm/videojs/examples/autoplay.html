Autoplay example

NEVER ASSUME AUTOPLAY WILL WORK.
USING MUTE ATTRIBUTE IMPROVES THE CHANCE THAT AUTOPLAY WILL SUCCEED
  • Apple has updated their autoplay policies to prevent autoplay with sound since Safari 11 release.
  • Chrome's autoplay policies was changed in April of 2018 and works as described in this linked article.
  • Firefox allow users to disable autoplay with a few configuration changes.
If you still insist to autoplay video, youcan check if certain browser allows it using the code below.

Example Player Code Setup

<!--Initialize player and plugin-->
<script>
	var player = videojs('player_one');

	// Hide big play button as quickly as possible
	var bpb = player.getChild('bigPlayButton');
	if (bpb) {
		bpb.hide();
	}

	player.nuevo({ 
		logotitle: 'Nuevo plugin for Videojs',
		logo: 'images/logo.png',
		logoposition: 'RT',
		logourl: 'https://www.nuevolab.com'
	});
	player.ready(function() {
		var promise = player.play();

		// If no promise is returned, show the BPB immediately.
		if (promise === undefined) {
			bpb.show();
    
		// If a promise is returned, wait until it either succeeds or
		// fails to show the BPB.
		} else {
			promise.then(function() {
				bpb.show();
			}, function() {
				bpb.show();
			});
		}
	});
</script>
