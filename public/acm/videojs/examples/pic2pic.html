Pic2Pic Plugin Example

Play video and Scroll down to see the minimized player.

Code Source

<!--Load videojs and plugins-->
<script type="text/javascript" src="../video.min.js"></script>
<script type="text/javascript" src="../nuevo.min.js"></script>
<script type="text/javascript" src="../plugins/videojs.pic2pic.min.js"></script>

<!--Initialize videojs and plugins-->
<script type="text/javascript" src="../video.min.js"></script>
<script type="text/javascript" src="../nuevo.min.js"></script>
<script type="text/javascript" src="../plugins/videojs.pic2pic.min.js"></script>
<script>
	var player = videojs('player_one');
	player.nuevoPlugin({ 
		logotitle:'Nuevo plugin for Videojs',
		logo: 'images/logo.png',
		logoposition:'RT',
		logourl:'https://www.nuevolab.com'
	}); 
	player.pic2pic();
</script>

Possible options

  • closeallow - set false to hide close button
  • mute - set true to mute minimized player
player.pic2pic({
	closeallow: false,
	mute: true
});
    Plugin works both: desktop and mobile device, is especially useful on website page with lot of content.
