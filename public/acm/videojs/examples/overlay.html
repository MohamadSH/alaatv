Basic Code Setup


<!--Load videojs and plugin-->
<script type="text/javascript" src="../video.min.js"></script>
<script type="text/javascript" src="../nuevo.min.js"></script>

<!--Initialize videojs and plugin-->
<script>
	var player = videojs('player_one';
	player.nuevo({ 
		overlay: "/domain.com.overlay.html"
	}); 
</script>

Overlay is loaded from separate html file. Overlay content can be just anything, image, text, advertising code.
Overlay content is centered vertically and horizontally. Overlay content that exceeds player size will be hidden.
Example overlay.html file is located in "/examples/images" directory. It's simple image link.
